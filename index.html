Redis-Commander

Redis management tool written in node.js

What is Redis-Commander?

Redis-Commander is a web based Redis management too written in node.js.

Features

  • View Redis config information
  • View a list of keys as an expandable tree
  • View individual key values with paging support for lists/sorted sets
  • Edit key values inline
  • Run Redis commands from command-line interface

Install and Run

$ npm install -g redis-commander
$ redis-commander --port 8081 --redis-host localhost
